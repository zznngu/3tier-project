\# WAS Server Overview



\## Role

\- Handles business logic and API requests

\- Connects to MySQL DB server

\- Manages DB connection pool



\## Environment

\- OS: Ubuntu Server 22.04 LTS

\- Runtime: Java 17

\- WAS: Spring Boot (Embedded Tomcat)



\## Network

\- DB Server: Private network access only

\- Web Server: Receives requests from Nginx
